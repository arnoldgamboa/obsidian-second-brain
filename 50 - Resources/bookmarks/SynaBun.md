# SynaBun

**SynaBun** is an open-source (Apache 2.0) toolkit providing **106 MCP tools** for vibe coding and persistent AI memory. It gives AI agents a local, private memory layer that persists across sessions and projects.

## Key Features

- **Persistent AI Memory** — AI remembers bugs fixed, patterns discovered, and decisions made across sessions
- **Recall by Meaning** — Natural language search; no exact keywords needed
- **100% Private & Local** — Memory stays on your machine
- **Any Model via MCP** — Works with any AI model through MCP protocol
- **Instant Memory Sharing** — Share memory spaces between tools/agents
- **Visual 3D Explorer** — Browse memories in a living, explorable knowledge space
- **Open Source** — v2026.5.3, Apache 2.0 license

## Why It Matters

Without memory, every AI session starts from zero. SynaBun solves the "context loss" problem in vibe coding by making AI memory personal, local, and persistent — similar to how human memory works.
