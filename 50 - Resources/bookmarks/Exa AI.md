# Exa AI

A search API designed for building research-based applications.

## What It Is

An AI-native search API that returns structured, relevant results for building apps that depend on high-quality search and research capabilities.

## Key Features

- AI-native search (not just keyword matching)
- Structured results suitable for app integration
- API-first design

## Why It Matters

A good option if building an app that is search-result based. Could be relevant for [[YouPastor-PRD|YouPastor]] if theological research features need external search integration, or for future products requiring research workflows.
