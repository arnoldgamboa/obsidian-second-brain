# Superpowers — Agentic Skills Framework

A complete software development methodology for coding agents using skill-based workflows.

## What It Is

A framework and set of conventions for building software with AI coding agents. Emphasizes skill-based decomposition, test-driven development, and structured subagent workflows.

## Key Features

- **Skill-based workflows:** Brainstorming → Design Doc → Implementation Plan → Subagent-Driven Development → TDD → Code Review → Branch Finishing
- **Writing-skills skill:** A meta-skill for creating new skills
- **Multi-tool support:** Claude Code, Codex, Cursor, OpenCode, Gemini CLI, Copilot
- **168k stars** on GitHub

## Why It Matters

Directly relevant to the skills/agents architecture of the PA system and could inform how [[YouPastor-PRD|YouPastor]] is built using agentic coding patterns.
