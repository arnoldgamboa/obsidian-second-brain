Bookmarked for future reference.

## Notes

- Atom Eve describes itself as “an AI agent for every job.”
- It provides a catalog of 28 agent templates across engineering, growth, revenue, support, operations, and data.
- Each agent is presented as a folder with a `README.md`, `SETUP.md`, and source, designed to be forked and contributed through GitHub.
- Potentially useful as a reference library for practical, bounded agent workflows—especially SEO, QA, analytics, content, support, documentation, and operations automation.
