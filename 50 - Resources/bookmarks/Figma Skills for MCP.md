# Figma Skills for MCP

Pre-built AI skills for the Figma MCP server that enable AI assistants to design and implement UI.

## What It Is

A set of pre-built skills for Figma's Model Context Protocol (MCP) server, allowing AI agents to interact with Figma programmatically.

## Key Skills

- `figma-use` — Write to the Figma canvas
- `figma-implement-design` — Convert designs to code
- `figma-create-design-system-rules` — Generate design system rules
- `figma-generate-library` — Connect components to code

## Why It Matters

Could automate UI/design workflow for projects like [[YouPastor-PRD|YouPastor]] or any product requiring rapid design-to-code iteration.
