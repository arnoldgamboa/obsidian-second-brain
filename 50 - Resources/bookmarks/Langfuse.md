# Langfuse

Open-source LLM engineering platform for debugging, tracing, and iterating on LLM apps.

## What It Is

A platform that provides observability, prompt management, and evaluation for LLM applications.

## Key Features

- **Observability:** Traces, sessions, agent graphs
- **Prompt Management:** Versioning, playground, experiments
- **Evaluation:** Datasets, LLM-as-judge scoring
- **Self-hostable** with Python/JS SDKs
- **OpenTelemetry-based** instrumentation

## Why It Matters

Useful for building and maintaining LLM-powered features where you need visibility into how the model is behaving, especially for agentic workflows like [[YouPastor-PRD|YouPastor]] or any app using the [[Superpowers — Agentic Skills Framework|Superpowers]] methodology.
