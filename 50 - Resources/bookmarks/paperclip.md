# Paperclip

URL: https://github.com/paperclipai/paperclip

Open-source control plane for coordinating teams of AI agents around business goals.

## Notes

- Paperclip combines a Node.js server and React UI for bringing your own agents, assigning goals, and tracking work and costs from one dashboard.
- It provides org charts, task and ticket management, scheduled heartbeats, budgets, governance and approvals, persistent agent state, audit trails, and multi-company isolation.
- It supports agents and runtimes such as OpenClaw, Claude Code, Codex, Cursor, Bash, and HTTP bots.
- Potentially useful as a reference for OperatorWire and other multi-agent workflows—especially goal alignment, delegation, persistent context, budget controls, and human oversight.
