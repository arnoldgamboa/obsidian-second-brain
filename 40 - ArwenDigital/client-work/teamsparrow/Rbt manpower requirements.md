<span style="font-size:15.428571701049805pt;">Manpower Request</span>
<span style="font-size:15.428571701049805pt;">1. Requesting unit to request a personnel in the system (Manpower Request Module)</span>
<span style="font-size:15.428571701049805pt;">2. If the position is not budgeted, the system will not allow the user to continue (Let us maintain a manpower budget module)</span>
<span style="font-size:15.428571701049805pt;">3. If budgeted, the system will send an email to the approving authority to approve the request</span>
<span style="font-size:15.428571701049805pt;">4. Once approved, system will send email to HR for the approved manpower request</span>
<span style="font-size:15.428571701049805pt;">5. The approved manpower request should be tagged to the position being sourced</span>

<span style="font-size:15.428571701049805pt;">Sourcing:</span>
<span style="font-size:15.428571701049805pt;">1. To follow the job street process for application</span>

<span style="font-size:15.428571701049805pt;">Recruitment and Hiring:</span>
<span style="font-size:15.428571701049805pt;">1. Once the applicant applies or submits his/her resume, the system will store all applicants data in the manpower pool</span>
<span style="font-size:15.428571701049805pt;">2. HR will review the manpower pool and selects applicants to process</span>
<span style="font-size:15.428571701049805pt;">3. Once selected, HR user will be able to schedule this for interview or testing</span>
<span style="font-size:15.428571701049805pt;">4. Email notification to the applicant and the interviewee</span>
<span style="font-size:15.428571701049805pt;">5. If applicant takes the test, user should be able to encode results and can tick if the applicants passed the testing</span>
<span style="font-size:15.428571701049805pt;">6. Once the applicant passed all and ready for hiring, HR user should be able to select the applicant and hire this</span>
<span style="font-size:15.428571701049805pt;">7. once hired, the system will update the 201 module (interface)</span>
