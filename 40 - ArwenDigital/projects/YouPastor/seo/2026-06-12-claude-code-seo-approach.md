# Claude Code SEO approach for YouPastor

## Context

Video: **Claude Code SEO: My New Site Just DESTROYED GOOGLE (Steal This)** by Income Stream Surfers.

The full transcript was not retrievable because YouTube blocked transcript access from the server, but the video metadata/description gave the core workflow.

## Core principle

Use **AI agents + real keyword data + website context** to create highly targeted SEO pages for specific search intent.

This is not generic AI blog-post creation. The method is to find actual search demand, connect the AI tool to the website/codebase, and generate focused pages that match a concrete user intent.

> AI + real keyword demand + site context + focused landing pages = scalable SEO pages that can rank.

## Video workflow

1. Connect the website to Claude Code through a connector.
2. Use keyword research tools like Semrush MCP or Google Ads Keyword Planner to find strong keyword opportunities.
3. Give Claude Code the site context and keyword data.
4. Have Claude Code synthesize the research into a focused SEO page.
5. Publish the page.
6. Repeat across related intent clusters until the site builds topical coverage.

## Tools mentioned

- **Claude Code** — AI coding agent used to inspect/edit/generate site pages.
- **Website connector / MCP connector** — bridge between Claude Code and the website/codebase/data.
- **Semrush MCP** — paid SEO/keyword/competitor data.
- **Google Ads Keyword Planner** — free keyword research and search volume ideas.
- **HarborSEO** — creator’s SEO product/tool.
- **Bright Data** — scraping/proxy/data collection tool for SERP/data collection at scale.
- **Ballinderry Park page** — example output page used in the video.

## Prioritization model

Score each keyword/page opportunity from 1–5:

| Factor | Question |
|---|---|
| Search intent fit | Is the searcher likely to want YouPastor? |
| Product fit | Does YouPastor solve this directly? |
| Conversion value | Would this visitor plausibly become a user? |
| Competition | Are current results weak/generic? |
| Page effort | Can we create a strong page quickly? |
| Internal authority | Can existing pages link to it naturally? |

Prioritize the highest combined score.

## Recommended current tool stack

Use existing/free tools first:

1. **Google Ads Keyword Planner** — export keyword ideas/search volume.
2. **Hermes** — cluster keywords, score opportunities, draft briefs/copy, edit repo/site if available.
3. **Manual/browser SERP checks** — judge real competition and search intent.
4. **Google Search Console** — validate impressions/clicks after publishing.
5. **Semrush later** — useful once faster competitor/keyword-difficulty workflows are needed.
6. **Bright Data later** — only if scalable SERP scraping/rank tracking is needed.
7. **HarborSEO optional** — evaluate only if it saves enough time over a custom Hermes workflow.
