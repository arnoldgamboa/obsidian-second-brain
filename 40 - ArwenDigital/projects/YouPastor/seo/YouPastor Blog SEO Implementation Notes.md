# YouPastor Blog SEO Implementation Notes

## What to add

### 1) Add a byline / author section
Include a clear byline on blog posts so readers and search engines can see who is writing.

**Suggested author details**
- Arnold Gamboa
- Engineering manager + pastor context
- Ministry/workflow expertise
- Links to relevant profiles

Why this matters:
- builds trust
- strengthens E-E-A-T
- makes the content feel more credible and human

### 2) Tighten long titles
Some titles are doing too much and should be shortened for better SERP display.

**Examples to trim**
- How to Turn a Sermon Into a Blog Post Without Losing the Heart of the Message
- A Midweek Devotional Template for Pastors Who Need Something Helpful by Wednesday
- Small Group Questions for a Sermon: A Simple Template That Keeps Discussion Moving

**Rule of thumb**
- keep the main keyword near the front
- aim for ~50–65 characters when possible
- let the description carry the extra detail

### 3) Keep the implementation focus
Use this note as a reference when patching the blog templates and post files.

## Suggested implementation order
1. Add author/byline metadata
2. Shorten the long article titles
3. Add updated dates
4. Create unique OG images
5. Expand the core posts for depth
6. Build pillar/category pages
