## Project Template

When starting a new project, create a folder in `brain/1-projects/` and a main note with this structure:

```markdown
---
tags: [project]
date: YYYY-MM-DD
status: active
---

# Project Name

## Goal
What are we trying to achieve?

## Status
- [ ] Milestone 1
- [ ] Milestone 2

## Related
- [[Areas MOC]]
- [[Resources MOC]]
```
