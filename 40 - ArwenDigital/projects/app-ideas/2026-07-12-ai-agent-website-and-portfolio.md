# Build AI agent website and portfolio

<!-- brain:upper-deck -->

> [!summary] Current truth
> Build an AI-agent services website and use two or three tightly scoped small-business pilots to create measurable case studies that can become repeatable paid offers.

## Current State

This idea has been filed in the ArwenDigital app-ideas incubator. The intended offer and pilot businesses have not yet been selected.

## Decisions

- Pilot agents must be bounded, measurable, and reusable.
- Free pilots should define scope, timeline, success measures, and testimonial or case-study expectations.
- The website should communicate the operating offer rather than present generic AI capabilities.

## Open Loops

- [ ] Define the target customer, positioning, and core offer.
- [ ] Choose two or three bounded small-business agent use cases.
- [ ] Identify pilot businesses.
- [ ] Define the free-pilot agreement and measurement plan.
- [ ] Build the website around the selected offer.
