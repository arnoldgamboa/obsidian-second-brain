# Build AI agent website and portfolio

## Raw thought

I need to build my AI agent website, then offer two to three free agents to small businesses as a portfolio-building process.

## Possible next actions

- Define the website's target customer, positioning, and core offer.
- Choose 2–3 bounded agent use cases that solve clear small-business problems.
- Identify 2–3 pilot businesses and define a free-pilot agreement, scope, timeline, and testimonial/case-study expectations.
- Build the website around the offer and use the pilots to create measurable case studies.

## Working principle

Keep each free agent tightly scoped, measurable, and reusable so the portfolio work can become a repeatable paid offer.
